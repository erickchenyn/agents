## 执行流程

1. **调用脚本检查工作区状态**
   - 使用 Bash 工具调用 `workspace-check.sh` 脚本（位于 `~/.claude/skills/workspace/` 目录）
   - 脚本会自动检查所有非主工作区的状态

2. **分析每个工作区的安全性**
   - 检查是否有未提交的更改
   - 检查是否有未推送的提交
   - 检查是否有打开的 PR
   - 检查远程分支状态

3. **显示工作区状态表格**
   - 工作区名称和对应分支
   - 安全状态（✅ Safe to clean 或 ⚠️ Not safe）
   - 统计摘要信息

4. **提供后续操作建议**
   - 如有安全的工作区，建议使用 `wclean` 清理
   - 显示总数、安全数量、不安全数量的统计

## 使用场景

- 查看当前有哪些工作区
- 了解哪些工作区可以安全删除
- 在执行 `wclean` 之前检查状态
- 工作区管理和维护
